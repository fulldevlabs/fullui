Theming is done with CSS variables. Either custom colors or Radix colors can be used.

## Colors

You set the color themes using CSS variables variables:

```css
:root,
.theme-light {
  --base-background: ;
  --base-background-surface: ;
  --base-background-subtle: ;
  --base-background-soft: ;
  --base-background-hover: ;
  --base-background-active: ;
  --base-border: ;
  --base-border-outline: ;
  --base-border-hover: ;
  --base-solid: ;
  --base-solid-hover: ;
  --base-solid-foreground: ;
  --base-text: ;
  --base-text-contrast: ;

  --brand-background: ;
  --brand-background-surface: ;
  --brand-background-subtle: ;
  --brand-background-soft: ;
  --brand-background-hover: ;
  --brand-background-active: ;
  --brand-border: ;
  --brand-border-outline: ;
  --brand-border-hover: ;
  --brand-solid: ;
  --brand-solid-hover: ;
  --brand-solid-foreground: ;
  --brand-text: ;
  --brand-text-contrast: ;
}
```

### Dark theme

Dark mode is set via a simple `.theme-dark` class.

If you want to use a dark theme by default, remove the `:root` selector from the light theme and add it to the dark theme.

Components in Fulldev UI that support theme switching have the `theme` prop available, which adds either the `theme-dark` or `theme-light` class

```css
.theme-dark {
  /* Same variables here as above */
}
```

### Radix colors

[Radix color palettes](https://www.radix-ui.com/colors) can be dropped and are automatically mapped to the right values.

The only thing you should do it change the color name, like `gray` or `blue`, to either `base` or `brand`. And again, place them in the `:root` / `.theme-light` / `.theme-dark` selectors.

You can even generate custom theme in no-time with [this awesome tool](https://www.radix-ui.com/colors/custom).

```css
:root,
.theme-light {
  --base-1: ;
  --base-2: ;
  --base-3: ;
  /* The rest of your radix palette... */
}
```

## Radius

At the moment only the border-radius is customizable, the rest should be done via regular css or Tailwind. We plan to add more theming options.

The `--border-radius` variable is a multiplier applied to all radius values. The default value is `1`.

```css
:root {
  --border-radius: 1;
}
```
