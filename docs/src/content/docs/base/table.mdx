## Props

The `Table` component exists of multiple sub-components. Each sub-component has its own set of props.

#### `<Table>`

<PropsTable props={frontmatter.props.table} />

#### `<TableHeader>`

<PropsTable props={frontmatter.props.tableHeader} />

#### `<TableBody>`

<PropsTable props={frontmatter.props.tableBody} />

#### `<TableRow>`

<PropsTable props={frontmatter.props.tableRow} />

#### `<TableCell>`

<PropsTable props={frontmatter.props.tableCell} />

#### `<TableFooter>`

<PropsTable props={frontmatter.props.tableFooter} />

## Examples

### `size`

<SizeProp />

```astro live
---
import { Table } from 'fulldev-ui'

const table = {
  header: ['Product', 'Quantity'],
  body: [['Apple', '2']],
}
---

<Table size="sm" {...table} />
<Table size="md" {...table} />
<Table size="lg" {...table} />
```

### Props vs slots

```astro
<Table>
  <TableHeader cells={['First Name', 'Last Name', 'Age']} />
  <TableBody>
    <TableRow cells={['John', 'Doe', '31']} />
    <TableRow cells={['Jane', 'Doe', '29']} />
  </TableBody>
  <TableFooter cells={[{ text: 'Total', colspan: '2' }, '2']} />
</Table>
```

instead of creating each row with a `TableRow` component, you can use the `rows` prop in the `TableBody` component.

```astro
<TableBody
  rows={[
    ['John', 'Doe', '31'],
    ['Jane', 'Doe', '29'],
  ]}
/>
```
